# Security Policy

## Reporting a Vulnerability

This project is a collection of text prompts and writing guidance for Claude Code. It contains **no executable code**, no server, no API, and no network-facing components.

If you discover a security concern related to prompt injection or any other issue, please report it by opening a GitHub Issue with the label `security`.

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Scope

- **In scope**: Prompt injection vectors, content that could cause harmful outputs when loaded by Claude Code
- **Out of scope**: Issues in upstream skills, issues in Claude Code itself, issues in AAAI Author Kit
